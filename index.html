📋 系统概述

Cedric 进销存管理系统 V2.0 面向中小型制造工厂、批发零售企业的智能库存管理桌面软件。支持成品、辅料、包材、原材料四大类物料管理。

适用场景： 中小型制造工厂 · 批发零售店铺 · 电商仓储 · 个体商户 · 多岗位协作

📘 操作指南

1. 登录系统

  • 输入用户名和密码登录（默认 admin / admin123）
  • 服务器地址留空为本机，客户端需输入主机IP

2. 库存管理

  • 点击顶部标签切换：成品 / 辅料 / 包材 / 原材料
  • 统计卡片显示种类、库存数量、库存金额、低库存预警
  • 支持按名称、规格、材质、颜色筛选物料

3. 新增物料

  • 填写物料信息，支持上传物料图片
  • 选择供应商或手动输入

4. 入库操作

  • 搜索并选择物料 → 输入数量 → 选择日期和备注 → 保存
  • 操作完成后可立即打印单据

5. 出入库记录

  • 支持按仓库类型、日期范围筛选
  • 每条记录支持编辑、打印、删除
  • 支持导出CSV文件

6. 月度结存汇总

  • 选择仓库类型和月份查看汇总数据
  • 支持导出CSV和打印

7. 呆滞料预警

  • 自动识别长期未流动物料，三级预警
  • 可自定义阈值天数并持久化保存

8. 物料详情

  • 点击物料行的「详情」查看完整信息和大图

9. 供应商管理

  • 供应商信息维护，支持增、删、改、查

10. 用户管理

  • 新增/编辑/删除用户，为用户分配角色

11. 角色权限管理 V2.0

  • 创建自定义角色，可视化勾选22项权限
  • 财务权限（单价/金额/统计卡）可分别控制

12. 操作日志 V2.0

  • 自动记录所有用户操作，便于审计追溯

13. 数据备份 V2.0

  • 一键下载数据库备份文件
  • 恢复：将备份文件覆盖到 %APPDATA%\inventory-system\inventory.db

🌐 局域网多用户部署 V2.0

角色操作说明
主机正常启动程序即可，服务端自动运行
主机查IP命令行输入 ipconfig，找到 IPv4 地址
客户端登录界面输入主机IP，再输入账号密码
数据存储所有数据统一存储在主机上
⚠️ 注意： 主机需保持开机，客户端才能连接。主机和客户端需在同一局域网。

❓ 常见问题

问题解答
仓库名称可以改吗？仓库名称可以根据具体的需求定制更改。
数据保存在哪里？SQLite 数据库文件，位于 %APPDATA%\inventory-system\inventory.db
如何备份数据？使用系统内置的「数据备份」功能一键下载，或手动复制 inventory.db 文件。
如何切换仓库类型？点击库存管理页面的标签页（成品/辅料/包材/原材料）。
软件需要联网吗？单机使用完全离线；局域网多用户需在同一网络。
支持哪些系统？目前支持 Windows 7/10/11。
客户端如何连接主机？主机查IP → 客户端登录界面输入IP地址 → 输入账号密码登录。
如何控制用户权限？超级管理员在「角色权限」中创建角色并勾选权限，在「用户管理」中为用户分配角色。
单价/金额显示***？当前用户角色没有「查看单价」或「查看金额」权限，联系管理员授权。
